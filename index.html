Was ist Bootstrap?

Bootstrap ist ein kostenkoses System, welches die Webentwichlung bedeutend einfacher und schneller macht. Bootstrap funktioniert über HTML eingebundene Links. Durch das zuteilen der bestimmten Klassen an die Elemente werden bestimmte Aktionen mit dem Element ausgeführt.


Was ist besonders an Bootstrap?

Der grosse Vorteil an Bootstrap ist, dass es responsiv ist. Es passt sich automatisch an dieBildschirmgrösse an. Die neuste Version von Bootstrap ist Bootstrap 4, die mit allen modernen Browsern kompatibel ist. Probieren Sie es selber aus, indem Sie diese Webseite zusammen ziehen.

Was ist Bootstrap?

Bootstrap ist ein OpenSource Framework welches von Mark Otto und Jacob Thornton aus dem Twitter Team entwickelt wurde. Es ist eine Kombination aus HTML, CSS und JavaScript und wird zur Darstellung von Benutzeroberflächen im Browser verwendet.


Was ist besonders an Bootstrap?

Der grosse Vorteil an Bootstrap ist, dass es responsiv ist. Es passt sich automatisch an die Bildschirmgrösse an. Die neuste Version von Bootstrap ist Bootstrap 4, die mit allen modernen Browsern kompatibel ist. Probieren Sie es selber aus, indem Sie diese Webseite zusammenziehen.

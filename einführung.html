Was ist Bootstrap?

Bootstrap ist ein kostenkoses System, welches die Webentwichlung bedeutend einfacher und schneller macht.
Bootstrap funktioniert über im HTML eingebundene Links, die die Anweisungen der Klassen beinhalten.
Durch das zuteilen der bestimmten Klassen an die Elemente werden bestimmte Aktionen mit dem Element ausgeführt.

Was ist das besondere an Bootstrap?

Der grosse Vorteil an Bootstrap ist, dass es responsiv ist. Es passt die Seite also automatisch an die Bildschirmgrösse an.
Dies macht Bootstrap sehr einfach. Die neuste Version von Bootstrap ist Bootstrap 4, die mit allen modernen Browsern kompatibel ist.
